Built for Experimenters

Framework First

BOF isn't just a toolkit—it's a foundation. Designed for extensibility from day one, it lets you customize components or add your own with ease.

JSON-Defined Questionnaires

Write portable, logic-aware surveys in JSON. BOF interprets both the questions and the calculations you define for them.

Multi-Condition Studies

Effortlessly define and deploy multi-condition experiments, complete with conditional routing and logic-driven task flows you can define via Python scripting.

Reusable Modules

Design once, use forever. BOF encourages modular development so you can reuse tasks, pages, or questionnaires across projects.

How It Works

Define Questionnaires in JSON

Write portable JSON surveys with logic and scoring rules built in.
Embed Custom Tasks

Integrate HTML-based tasks like Unity games or custom JS components.
Define Data Schema

Describe experiment outputs using a simple JSON-based schema definition.
Send Data via POST

Send structured task results back to BOF using simple POST requests.
Live Dashboard

Track real-time participant progress from a built-in monitoring view.
Export Results

Export wide-format CSV files or download the complete SQLite dataset.
